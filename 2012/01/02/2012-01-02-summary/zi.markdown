今天我在twitter上发了[一条推](https://twitter.com/DrazenGY/status/287225534709112832),原本只是娱乐的一条twitter，却引起了同事们关于挺方舟子和倒方舟子的争论。我之前也一直关注着方舟子，从老罗和方舟子的矛盾，到后来方舟子反中医、赞成转基因、倒韩寒，很多人分成了两派，要不把方舟子当成神，要不把方舟子当成彻头彻尾的骗子。我在这里说一下我的观点。
首先我想就事论事，来看看方舟子的观点到底是怎样的。

### 1. 和老罗
这里面参杂了太多私人的感情，具体的内幕除了当事人，估计谁也不清楚，也就不用争论了。不过我确实佩服老罗，有信念，能坚持。 

### 2. 关于中医
目前来说我是反对中医的，尤其是打着中医的幌子进行养生骗钱的。我反对中医基于以下几个理由：

* 药物风险。通常中医开的第一副药方，不会有什么作用。如果你想通过中医调理，一般要找一个固定的中医连续吃一年的药才有一些效果，这也是很多人说中医疗效慢。一个物体的作用是两面性的，中药在调理身体的时候也在人体内慢慢积累毒素。中药见效慢，同样毒性发作的也慢，在我看来喝中药就是在和慢性毒药。  
  尤其是药方基本上都是靠医生的水平来开的，一个医生的好坏基本上就是在药方上体现出来。如果你一不小心碰见一个庸医，那你的身体就成了他的实验场，他会用N副草药来试探你的身体，你的身体发生了什么谁也不知道。而西医在这方面做得比较好，一个药物的面试要经过很长时间的临床试验来确保安全，虽然不可避免的会发生一些事故（抗生素的滥用），但已经在很大程度上控制了风险。
* 成品药。中医只是历史长，但时间长并不代表着先进。中医发展到现在，真正能制成成品药，并被广泛接受的也没有多少，去药房看一下就知道了。另一方面，在以中医为主的漫长的中国历史中，中国人的平均寿命却没有明显的延长，反而是在引进西医、医疗条件大幅提升之后，中国人的体制得到了改善。
* 养生。现在每次和妈妈打电话，妈妈总会叮嘱我一堆事情：嘴上长痘痘是胃虚火，不能吃凉的；跑步不能出汗；肝脏五行属金等等，我已不知如何应对。中医发展上千年，五行相克等等理论早已能够自圆其说，中医其实是人在长期自然发展中总结的一套基础规律，所以人身体上发生的事情都能够用中医解释的通，但中医并没有提供很好的解决方法。盲目的迷信中医只是害人害己

中医到现在也没有证明对人体确实有益，在我看来中医只是中华文明史上的一块遮羞布，用来满足我们的虚荣心，中医本质上和阿拉伯等地方的本土医术没有区别。普世的价值才是值得推广的。所以方舟子反对中医，我认为是在做一件好事。

### 3. 转基因
我认为现在判断转基因是好是坏没有事实依据，我采取不反对，不食用的态度。所有的食品标明转基因还是非转基因就好。支持转基因的，比如方舟子，让他吃个几十年，如果没有问题再推广。最重要的是我们的知情权和选择权。

### 4. 关于韩寒
我喜欢韩寒的文章，是因为我认同他文章中的表明的态度，无论是《像少年啦飞驰》《长安乱》还是《韩三篇》，至于他是不是抄袭的无所谓，时间会证明一切的。抄袭代写的行为，最终会受到惩罚，如果仅仅凭借着一些臆想的，或者咬文嚼字的证据来证明抄袭，我觉得很无聊。

比如你在我的这篇文章中会发现一些错别字，还有一些“的得”不分的词语，有的时候用的对，有的时候用的错，完全是不小心没看到或者输入法的问题，如果有一天我成了知名作家，有人用这篇文章来说明我是抄袭的，我要如何自证？别人说我是太监，我就要脱了裤子让别人看我的小JJ吗？谁没有年少轻狂的时候，偶尔说错一两句话、自大一下而已，在韩寒这件事情上，我觉得方舟子无理取闹。

###5. 关于方舟子
在方舟子不断质疑别人的时候，也有人在质疑方舟子，主要集中在两件事情上：刘菊花的文凭真假和方舟子打假基金的帐务问题。我欣赏方舟子的执着，但对于方舟子这种只挑别人毛病，逃避自身问题的做法很是鄙视。自己的人品都不可靠，如何证明自己做的事情是对的？？？即使你是正确的，我也会抱有怀疑的态度，因为你的人品摆在那里。


----------

任何人的行为都是有一定目的性的，经济学上叫利己行为。所以不论老罗、方舟子或者韩寒做出什么行为，他们都是有目的的，或许是为了他人的认同，比如老罗，或许是出于经济目的，比如方舟子支持转基因（国际粮食巨头是支持转基因的，便宜、量大，无论是食用还是工业用都是满足他们的利益的，不过纯属我的参考维基之后的猜测，当不得真），或者出于自我价值实现，比如韩寒。比如我，写这篇文章，也是想给博客带点流量。

很多时候我们是先给一个事物定了性，然后带着有色眼镜去看一件事情：我支持韩寒，所以方舟子说的都是错的；方舟子反中医是对的，所以方舟子是个好人，所以韩寒肯定抄袭了，因为方舟子说韩寒抄袭了。

你是挺方还是倒方？
