发表于 2023-08-28   |   分类于 Work   |  

之前一直想做Vlog博主，但是没想好做哪一方面的，倒是在各个视频网站开了账号，周末的时候看了一些视频，突然想到我可以结合我的账号名称：「钢铁是怎样没有练成的」做一系列躺平方向的视频，当然视频来源主要是各个视频平台上的第三方账号，我通过下载、二次编辑、转发等方式来发布，唯一的担心就是版权问题了，不过我应该会在视频上标注好来源，或许仍然会被投诉，不过我想我主要的发布平台应该是在youtube上面，加上英文的字幕（当然是通过AI了），毕竟国内的平台除了带货，没有其他方式变现。

我想会有以下几个主题的视频：

多锻炼

基本上就是关于健身、健康饮食的搞笑视频，不会真的去锻炼，而是通过反讽的方式去躺平。

交益友

关于朋友的搞笑视频，全部是「损友」，没有益友。

街（gāi）溜子

主打City Walk：

  1. 当下比较火的主题；
  2. 之前做Carlog的时候，就想做一个汽车行驶在各个城市的视频，作为白噪音还是挺好玩的，citywalk正好切合这个主题；

努力工作

当然是整顿职场了。

省吃俭用

买买买，各种奇葩物品，网红探店，主打的就是「宫廷玉液酒，一百八一杯」。

其他的以后发现了再增加吧，我觉得除了街（gāi）溜子这个系列是原创，其他的基本上都是搬运，另外再多增加一些tag，方便推广。

在 Macbook 上运行 ChatGLM-6B

发表于 2023-08-17   |   分类于 Code   |  

克隆 ChatGLM-6B 仓库

git clone https://github.com/THUDM/ChatGLM-6B

创建虚拟环境

cd ChatGLM-6B
python3 -m venv .venv

激活虚拟环境

source .venv/bin/activate

安装依赖

pip install -r requirements.txt
# 安装 PyTorch， 使用 MPS 后端来在 Mac 的 GPU 上运行
pip install --pre torch torchvision torchaudio --extra-index-url https://download.pytorch.org/whl/nightly/cpu

下载模型

cd ..
git clone https://huggingface.co/THUDM/chatglm-6b chatgml-6b-model

修改代码使用 MPS 后端和下载的好的模型

修改 webapp.py 文件。

  1. 将 .cuda() 改为 .to("mps")
  2. 将模型和 tokenizer 的路径改为 huggingface 上克隆下来的 chatglm-6b 仓库本地路径。下面我的例子中，本地路径为 /Users/roy/git/chatgml-6b-model。

-tokenizer = AutoTokenizer.from_pretrained("THUDM/chatglm-6b",trust_remote_code=True)

-model = AutoModel.from_pretrained("THUDM/chatglm-6b",trust_remote_code=True).half().cuda()

+tokenizer = AutoTokenizer.from_pretrained(
+ "/Users/eson/git/chatgml-6b-model", trust_remote_code=True
+)
+model = (
+ AutoModel.from_pretrained(
+ "/Users/eson/git/chatgml-6b-model", trust_remote_code=True
+ )
+ .half()
+ .to("mps")
+)

运行

`python webapp.py|

这将会在本地启动一个 web 服务，会自动打开浏览器。

这两年的反思

发表于 2023-05-29   |   分类于 Work   |  

21年7月20日，我加入了一个教育行业创业公司，担任产品总监，虽说当时国家刚刚打掉了一个行业（K12），但我还是对互联网教育行业充满了热情和希望，义无反顾的从斑马 跳槽到刚刚成立的创业公司，这次是认真的想要做一些事业，转眼将近两年过去了，公司有了以下重要的变化：

  1. 两年前疯狂招人，去年年初上海疫情刚开始的时候，员工数量巅峰将近60人，其中平台产品的研发团队就将近20人，从去年下半年开始经历了三轮裁员之后，现在只剩我一个人了；
  2. 公司一开始定位科技公司，去年开始自己制作课程内容，然后扩展出来了将近40人的内容制作团队，包括摄影、后期、音乐、调色等等，完全变成了一个重成本的内容生产公司；
  3. 很快烧完了第一笔的投资款（数千万），但是销售不给力，几乎等于零收入；

仔细想想，公司几乎犯了所有创业公司会犯的所有错误。

公司商业模式一开始就没想清楚

公司通过什么来盈利呢？卖课程吗？还是卖 SaaS 平台的账号？ 如果是课程的话，课程从哪里来？我们自己做课的成本和效率行不行？课程单价是多少？这些都没想清楚，只是想做普惠的互联网商科教育，正好有一个有耐心的投资人，就这么开始了。

公司的目标是什么

一直到公司创立一年半，消耗资金数千万的时候，我们都还不清楚公司的目标是什么？上市太遥远，年度目标总该有吧？基于年度目标的季度目标？以及相应的拆分路径？这些都没有…公司也没有讲明白愿景、使命、价值观，虽然听上去有些虚无缥缈，但却是统一大家想法和行动的工具。

花了太多不该花的钱

拍摄课程的设备都要买最好的，办公室选在外滩，租金贵的一比，很多时候方向不清楚的时候，做什么都是错的，造成人力、金钱、物力的浪费。如果保持较小规模团队不断试错，或许第一笔资金能撑五年也说不定。

等我想到了其他的问题，再写一篇吧。或者展开讲讲上面说的三点。
