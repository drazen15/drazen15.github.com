如何理财

很长一段时间内，我都是月光族，随意买各种电子产品，胡吃海喝，不断提升自己信用卡的额度，透支着属于未来的钱，直到去年结婚，买了房子，终于意识到自己要承担更大的责任了，想要理财的念头才开始浮现。
<!—- more —>
我给自己简单做了一个财务计划，年初的时候开始执行，到现在为止效果还不错：

  1. 准备至少可以应付半年的应急款（已经达成）；

  2. 还清所有的信用卡和亲友的欠款；

    目前还有买房的时候欠的一些外债，预计今年底可以还清；

  3. 把税后收入的20%拿来进行投资；

    • 理财，了解各种理财产品的门槛、风险，以及投资技术；
    • 对自己的长远投资，包括学习，参加培训，人脉的积累等等；

    • 购买保险（合理的风控）；

      对自己的投资主要是在学习上的，购买书籍、参加培训，其余的钱暂时都投入到了股市，后面会尝试一些定投、货币基金等；

  4. 尽快还清房贷；

    这个只能期望公司给力了…

  5. 增加被动收入在总收入中的比重（所谓被动收入就是不干活只拿钱的收入），多元化的收入；

  6. 合理规划自己的财务，减少不必要的支出；

最重要的是：不要轻易花掉你现在和未来的钱！

第3周小结

感觉自己最近负面情绪有点积压，工作中甚至出现了抱怨的现象。

我们公司作为一个创业公司，从去年的十来个人成长到今天的一百多人，我们又处于一个极其高速成长的行业里面，所以我们会遇到这样那样的问题，这也是不可避免的。

纯粹的目标

发表于 2014-08-21   |   分类于 Thinking

创业最怕的是什么？每个人都想太多。

简单的说，就是目标不纯粹，每个人做事情都会瞻前顾后。

举个简单的例子，比如说公司今年的目标是销售额一千万，分到你的部门的指标是一百万，这就是你的目标，所有的事情都应该围绕着这个目标来进行，每一件事情做不做的评判标准都应该是：对我们的目标有没有影响。
如果这个时候出现一个人或者一个事情，对目标的达成会造成负面影响，那我们就应该马上制止，并把团队拉回正规。如果你的目标不纯粹，那么你处理这些事情的时候就会考虑很多其他的因素，比如私人关系、个人利益等等，相对来说，你达成目标就会非常困难。

另外很多人执行力不够，总是找理由说想法太多不知道怎么做，其实就是没有想法不知道怎么做。

自勉。

比特素养

发表于 2014-05-01   |   分类于 Thinking

一天的时间读完了『比特素养』这本书，然后发现对我自己来说没什么用，我没有获取到任何有价值的信息，这并不是说这本书写的太烂，而是这本书中所介绍的方法论我都知道，并且已经在实践了。

这本书主要介绍了一个人如何在互联网时代处理大量的信息，从邮件、待办事项、阅读信息等各个方面教你一些方法，可以看做是一个方法论的合集。

在我看来每个人都应该有自己的方法论，有自己做事情的逻辑，然后不断完善这套方法论，提高自己的效率。对于没有自己方法论的同学，可以参考一下这本书，帮助自己建立方法论。

我的方法论很简单，核心思想与GTD的理念基本保持一致，分为四个阶段：收集、计划、执行、回顾。具体运用到实际中，比如阅读，是这样的：

  1. 收集：通过RSS、微博、知乎、Twitter等信息源，获取信息，简单筛选之后，将有用的信息保存在Pocket或者Evernote中。
  2. 计划：每天花固定的时间进行深入阅读。
  3. 执行：深入阅读之后，将自己的想法整理出来，一般会写Blog或者笔记。
  4. 回顾：每个一段时间之后（一般是一个月），重新阅读之前的Blog或者笔记，看自己的想法是否和之前的一样，如果不一样，哪里发生了改变，为何改变，不断更新自己的知识库。

这套方法用到其他的领域一样可行，我用这套方法管理团队，处理个人工作计划，进行财务规划。敏捷开发其实也是这样的流程，只不过处理具体事务的时候，会根据不同的目标，不同的情景，执行的形式或者细节会有很多不同。

如何正确的跑步

发表于 2013-07-24   |   分类于 Thinking

坚持跑步也有一段时间了,现在将自己的跑步经验总结一下,再结合知乎上一些跑步达人的分享,整理出这篇文章,即是之前的回顾,也是未来的一个指引.

跑步姿势

跑步姿势是最重要的,比跑鞋还要重要.良好的跑步姿势,能够让腿部形成一个天然的弹簧,不仅能保护膝盖,脚踝等关节组织不受冲击,还能极大的降低磨损.

简而言之,正确的跑步姿势为:

  1. 身体正直,跑步的时候微微前倾.
  2. 脚尖自然落地.
  3. 手臂向前摆动.手臂与肩膀向后扩展使胸部呼吸顺畅(可使用腹部呼吸)
  4. 臀部收在身体正下方.
  5. 头部向前,保持在肩膀正上方.

我之前跑步一直是脚后跟着地,后来发现前脚掌着地会让跑步更轻松.慢慢的我也就变成前脚掌着地了.

赌客信条

行为经济学建立在三个理论之上:前景理论,锚定效应和心理账户原理.其中<怪诞行为学>对锚定效应的解释比较全面.[赌客信条](http://book.douban.com/subject/4265573/)里面我更多关注的是前景理论和心理账户原理.

前景理论

前景理论有四个原理:

  • 确定效应:在确定的好处和赌一把之间,做一个选择,多数人会选确定的好处.
  • 反射效应:在确定的坏处和赌一把之间,多数人选择赌一把.
  • 损失规避:多数人对损失比对收益更为敏感.
  • 参照依赖:多数人对得失的盘对往往由参照点决定.
