学会汇报坏消息

发表于 2023-03-03   |   分类于 Book   |  

以下内容为《靠谱》一书的读书笔记，作者侯小强。

汇报坏消息的最终目的，是为了正视问题，以免风险持续蔓延，造成不可估量的损失，汇报坏消息最终是为了获得解决方案。

  1. 一定要坦诚，不遮遮掩掩，要把关键细节和可能的风险汇报清楚。
  2. 当面汇报更好，汇报之前要反复演练，思考周全。
  3. 如果有自己的责任，一定要真诚道歉，敢于承担，犯错就要认，挨打要站直。
  4. 一定要力所能及的给出结局方案，并且寻求上级的支持。
  5. 一定要让上级感受到你全力以赴解决问题的态度和决心。

如何看一个人可用不可用

发表于 2023-03-01   |   分类于 Book   |  

以下内容为《靠谱》一书的读书笔记，作者侯小强。

现在用人的成本真的太高了，所以面试以及试用期就很关键，不合适的赶快换掉。

看结果

事事有交代，件件有回应。一定要看确定性的结果。

看勤奋

引用长者的话：「一个人的命运当然要靠自己的努力，当然也要考虑历史的进程」，不努力什么都没有，懒人遇到贵人的机率几乎没有。

看学习能力

这个非常重要了，学习能力决定了上限，尤其是在发展速度非常快的互联网行业。

另外还有一点比较重要的就是是否有开放性的思维，怎么判断呢？就是通过批评来判断，如果能沟接受批评，说明还行。如果不能接受，总是辩解，抱怨，或者甩锅，这种人就不要重用，关键时刻会掉链子。

职场七问

发表于 2023-02-28   |   分类于 Book   |  

以下内容为《靠谱》一书的读书笔记，作者侯小强。

每天工作都要不断的问自己：

  1. 上级的目标是什么？
  2. 个人的目标是什么？
  3. 是否掌握了有助于目标达成的关键方法和关键资源？
  4. 是否把足够的时间用在了对行业和竞品的调查研究上？
  5. 是否清晰的了解你和团队每一个人的投入产出比？
  6. 是否将资源、认知、注意力压倒性的投入到最重要的事情上？
  7. 是否能够坚持每天复盘、优化策略？

其实就是时刻关注OKR，高频词的复盘，调整自己的投入，选择最高效实现目标的路径。

我的生产力工具

发表于 2023-02-14   |   分类于 Work   |  

2.14情人节，写一篇关于生产力工具的文章，最主要的原因是我没有车:)

Apple Macbook Pro 14’

刚刚买的新电脑，定制款，M2 Max的芯片，12核中央处理器，38核图形处理器，16核神经网络引擎，64G内存，1TB固态硬盘，基本上算是定制的次顶配，对我来说性能应该是过剩的，我想之后会用来做的事情：

  1. 视频剪辑，主要是自己的一些频道，微信视频号，youtube，B站等。
  2. 学习Python，写代码；
  3. 用一些虚拟机；

Apple Macbook Air 13’

公司发的笔记本，低配，只有8G的内存，256GB的硬盘。就平常写写文档，做做PPT还可以。

DJI Pocket

用的很少，希望今年能拍更多的嘟嘟，然后年底可以考虑换第二代。

HHKB HYBRID TYPE-S

目前用的键盘，还算是比较满意，主要是比较小，一开始的时候不太熟悉，因为键位有点奇怪，后来通过Karabiner调整之后已经非常舒服了。

Apple TrackPad

自从用了TrackPad之后就再也没用过鼠标了，已经买了有三四个，最近用的是黑色款，配上黑色的HHKB，简直完美。

Apple iPad Pro

买了两年的iPad Pro，现在基本上就做两件事情，听歌和看书。没有太发挥作用。

Microsoft Surface

这台电脑已经有五年了？某一年公司年会抽奖抽中的，现在主要是用来报税，开发票等业务，毕竟国家部门的很多业务都必须在Windows环境搭配上IE才能完成。机器本身还是挺好用的，在没有安装过多软件的时候，还能保持不错的流畅度。

INCASE ARC Day PACK

我太喜欢INCASE的包了，上一个INCASE的包，应该用了有小十年，当然没有一直在用。我用过Tuler，Tumi的各种包，还是INCASE的最轻，背着最舒服，设计最合理了。颜值也是杠杠的，非常喜欢。
