羊了个羊

Posted on 2024-07-11 In Game

刚才一不小心又通关了「羊了个羊」第二关。

这个游戏在当时也算是一个小奇迹了，忘记发布有多少年了，当时几乎全朋友圈的人都在玩。

这个游戏有几个特点：

  1. 第二关巨难，很少有人通关；
  2. 总共就两关，99%的玩家倒在第二关；
  3. 机制简单，就是三消，谁都能上手；
  4. 有人通关会炫耀，毕竟不是每个人都能通关；

我一度也以为这第二关是无法通关的，我也不信很多人发出来的秘籍、分析等等，我觉得第二关就是个噱头，直到我通关了第二关……

经过我这么长时间的游戏，几乎每天都会玩至少三局，每局的时间不会超过2分钟，我基本上很容易能判断出当天的关卡是否能通关：

  1. 完全不可能通关的，叠加层次多，每次可控元素少，尤其是出现草垛的时候，几乎不太可能通关；
  2. 多玩几次可以通关的，叠加状态没那么多，没有草垛；
  3. 两次以内可以通关的，叠加状态少，每次可控的元素多 ，没有草垛；

通过我这种简单的判断，每个月我基本上都能通关数次，虽然后面增加了很多道具、玩法什么的，但我还是忠于基本模式，每天玩几局，权当消遣，尤其是现在卸载了王者荣耀之后。

推荐指数：⭐️⭐️⭐️⭐️★，做为一个微信小游戏，短时间获得正反馈，有一定的难度，已经很不容易了，推荐！

Day 1000 之「再见」

Posted on 2024-07-09 In Life

今天想写的是「再见」。

如何和不想失去的人说再见，我到现在也没学会。

这三十多年，看书哭的最惨的一次就是看余华的《活着》，从那时候起，我就觉得「活着」两个字，是中国文化中最有力量的两个字。

神农尝百草，盘古开天辟地，国人随地种菜，还有很多很多的人和事物，告诉我们，遇到难事，不要放弃，不要放弃的意思就是先活着，活下去才有希望。

可是我们要如何说再见呢？

搞一场盛大的告别仪式？比如葬礼，或者说生前的葬礼，就如「非诚勿扰2」。

完成自己所有的遗愿，然后轰轰烈烈的死去，就像「 Stand Up Guys」（「兄弟出头天」）。

或者安安静静的离开？就像「重庆森林」里面的林青霞？

还是像「蒙古草原，天气晴」中的普洁，在某一个清晨的意外中悄悄走了。

又或者像是「再会长江」中的茨姆和冬冬，相隔十来年，再见已物是人非。

很多次都会想起「东京爱情故事」里面的赤名莉香和永尾完治，在一个背向而行的游戏中，始终只有一个人回头。

我没有和爷爷奶奶好好说再见，爷爷还见了最后一面，奶奶没有见到。

我没有和姥姥姥爷好好说再见，爸爸、舅舅和我从医院接回了姥爷，一路上谁都没有说话，回到老院子，姥爷就去世了。我也没见到姥姥最后一面。

我没有和所有的同学好好说再见，走的时候大家目送我背着行李给我喊加油，我头也没回，就这样十好几年没见了。

你说出的再见，不是再次相见，而是再也不见。

真的直到死的那一刻才能释怀吧，因为那是自己对别人说的再见，然后我们就头也不回的走了。

狗日的腾讯-王者荣耀2

Posted on 2024-07-04 In Game

最近太可乐了，虽然我没有继续玩游戏了，但是看磊哥对抗策划还是非常有意思的。

磊哥的思路就是，你不让我好好玩游戏，我就找你人机的漏洞，然后让大家一起恶心你：

  1. 通过特殊文字来勾引人机对话，特别翻译腔的回复，确定人机后再选择对应的打法；
  2. 继续科研装备，快速拆塔；

核心还是拆塔胜利，但是评分巨低，然后下一局给你匹配高手或者NB的人机；

然后王者荣耀策划的对策是什么呢？

  1. 训练更加高级的人机，本质上还有有一套人机思维模型；
  2. 封掉磊哥的账号，但是磊哥精神永存；
  3. 导流到其他游戏，谡下学院，或者说之后的王者荣耀世界；
  4. 说是平衡，其实更加不平衡，比如新版本的百里守约；

推荐指数：★，彻底结束了。

你到底想要招怎样的人？

Posted on 2024-06-26 In Work

最近几年经济形势不好，找工作相对来说要难很多。我工作十几年，写过很多JD，招过很多人，也应聘过很多公司（有的公司我觉得非常合适，结果简历筛选都没过）。

我一直有一点疑惑，招聘公司写出来的JD，有的很明显感觉是病急乱投医，为什么还要按照这个模子去套候选人？

比如最近两年出海业务比较热，有的公司招聘会写到希望有做过出海业务的经验，但实际上不是希望，而是必须。我所纠结的点在于：

  1. 候选人做过出海业务，所做的业务是否和你公司的业务匹配呢？
  2. 出海的地区是否匹配？
  3. 候选人是被迫出海还是真的热爱海外业务？
  4. 候选人出海必须，那是不是说明其他的条件优先级不高？比如学习能力，好奇心，适应性，抗压能力，逻辑能力？
  5. 候选人的能力模型里面，尤其是产品的能力模型，出海经验是第一优先？
  6. 一个候选人非常熟悉海外市场，用的工具产品都是海外开发者开发的，另一个候选人做过海外业务，但是自己不怎么用这些东西，选哪个？或许第一个根本进不了面试。

反正遇见这种公司，我是会非常谨慎考虑的。

除了出海，还遇见过大型软件公司招聘工业化产品，必须做过相关软件，做输入法软件的必须做过输入法，哪怕你自己设计过输入法也不行，做中台运营工具的必须做过很多运营工具，哪怕你是一个资深买家也不行。

不可理喻。

我招聘产品候选人，其实主要看：

  1. 学习能力是否足够强，主要看他的知识边界；
  2. 好奇心，是否足够关注现在的趋势；
  3. 执行力，项目的落地能力，包括跨部门沟通，向上汇报，向下宣贯；
  4. 过往工作经验是加分项，不代表能做好现在的业务。

可能就是按照我的模子提出来的候选人画像吧。

狗日的腾讯-王者荣耀

Posted on 2024-06-24 In Game

一直以来对腾讯都没有什么好感，所以准备做一个主题系列「狗日的腾讯」，来吐槽吐槽我做为一个普通用户对腾讯的很多想法。

首先就是王者荣耀了，这个游戏我已经玩了八年，先列一些数据：

  1. 游戏天数2578天（截止到今天为止）；
  2. 对战场次10432；
  3. 英雄数量117；
  4. 皮肤数量169；
  5. VIP8（花了大概6万还是6千？，一个夏侯惇的无限飓风号就2000多了，还有两个孙悟空的星元皮肤等等）；
  6. 综合得分71.5；
  7. 胜率50.9%；
  8. 全场最佳1162，败方最佳1295，三连决胜928，四连超凡119，五连绝世13，超神731，

几乎每天都要玩一两局，一开始会非常耐心的钻研技术，看教学视频，购买喜欢的皮肤，但是最近两年，完全看不懂王者荣耀是想要干嘛，最大的问题其实就是匹配机制的问题：

  1. 抢分路，一个老大难的问题，最近出了推荐分路才缓解这个问题，策划脑子吃屎了？
  2. 演员摆烂问题，抢经济；

尤其是现在磊哥破解了匹配机制之后，大家才发现，一直以为是在「玩游戏」没想到是「游戏玩自己」，匹配机制想让你赢，你摆烂都能赢，想让你输，你超神都能输，你以为你在玩5V5公平竞技游戏，其实你是1V9人机挑战赛。

随着AI的发展，一切都只会被玩弄于股掌之中。

这个赛季结束，AFK了。

推荐指数：★，已经完了，这个游戏已经结束了，即使后面再出王者荣耀宇宙，也没有任何希望了，磊哥的鸡爪流让所有用户有一次看到了「狗日的腾讯」不把用户当人看。
