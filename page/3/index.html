工作中最重要的能力是什么？

Posted on 2024-07-28 In Work

我认为工作中最重要的能力是解决问题的能力。

可能有人谁说这不是废话吗，重点是什么才是解决问题的能力？

我举个例子，公司最近招聘了两个实习生，上周让其中一个实习生负责公众号的某篇推文，给了她推文的方向，然后她反复修改了几版，所有人都不是很满意，在这个过程中她也提到了很多问题，比如：

  1. 推文中的素材在哪里呢？
  2. 行距要多少？
  3. 字号多大？
  4. 有没有推荐的模板？

其实上面的问题，她都可以自己去找到答案的，只是需要一点点耐心或者刨根问底的精神，但是她选择最简单的方法，把问题抛到工作群里面，期待有人直接给她答案。

工作中经常会碰到类似的事情，老板的工作要求只是一句话，你要怎么去完成呢？很多人说那是老板的问题，可是老板付给你工资啊。

所以我觉得解决问题的能力很重要，其中就包括了很重要的解决老板问题的能力：

  1. 知道工作的目标是什么，这个是可以提问的；
  2. 知道目标后，拆解目标，分析用户；
  3. 参考公司过往做法，或者行业内标杆的做法；
  4. 注意细节，注意细节，注意细节！
  5. 工作做到有理有据，尽量想到可能被问到的问题，不要被别人到问题问倒；
  6. 输出要规范；

我一时间想到的就这样，以后有想到的再补充。按照这个思路，做一篇公众号的推文起码不是什么难事吧？

盲人按摩

Posted on 2024-07-24 In Life

大街上的盲人按摩应该是分流派的，虽然都是一个江湖门派，练的功夫套路差不多，但是细究起来，就像是华山派的剑宗和气宗，盲人按摩也是有分支的。

有大力宗，讲究的是大力出奇迹，师傅多是男性，只有把客人摁的嗷嗷叫，师傅才觉得到位了，若是客人一声不哼，师傅会认为受到了侮辱，手上的力道不由增加几分。

有点穴宗，讲究的是以巧借力，主攻穴位，尤其是客人酸痛明显的穴位，这样即使不用很大的力，也可以引起客人酸痛的感觉，按一圈下来，也是能缓解不少，这个手法要求就比较高，第一要能找到疼痛的穴位，第二要用巧劲，用力大了客人疼，用力小了没感觉，一般来说这样的好师傅难找，找到一个就轻易不会再更换。

有科技宗，我更愿意称之为伪科学宗，我对这个流派是有点不屑的，主要的方式是拔火罐，艾灸，刮痧等，看上去效果显著，毕竟你的皮肤都烂了，实际上只是分泌了更多的多巴胺而已。我从来都十分抗拒这个流派。

我每到一个新的城市，就会去当地的盲人按摩，由于吃不透师傅的招式流派，一般只能先提出一个大概的范围：男的，力气大的。

上海的有宝山康园盲人按摩，原来的7号，现在的15号，都是点穴宗高手。苏州吴中东吴门诊部的25号，是我现在发现的点穴宗顶尖高手，由于这两周没有预约到，换了一个9号和19号，没想到是这个门店排名前两名的大力宗高手，把我摁的死去活来。

以后非25号师傅不去了。

羊了个羊

Posted on 2024-07-11 In Game

刚才一不小心又通关了「羊了个羊」第二关。

这个游戏在当时也算是一个小奇迹了，忘记发布有多少年了，当时几乎全朋友圈的人都在玩。

这个游戏有几个特点：

  1. 第二关巨难，很少有人通关；
  2. 总共就两关，99%的玩家倒在第二关；
  3. 机制简单，就是三消，谁都能上手；
  4. 有人通关会炫耀，毕竟不是每个人都能通关；

我一度也以为这第二关是无法通关的，我也不信很多人发出来的秘籍、分析等等，我觉得第二关就是个噱头，直到我通关了第二关……

经过我这么长时间的游戏，几乎每天都会玩至少三局，每局的时间不会超过2分钟，我基本上很容易能判断出当天的关卡是否能通关：

  1. 完全不可能通关的，叠加层次多，每次可控元素少，尤其是出现草垛的时候，几乎不太可能通关；
  2. 多玩几次可以通关的，叠加状态没那么多，没有草垛；
  3. 两次以内可以通关的，叠加状态少，每次可控的元素多 ，没有草垛；

通过我这种简单的判断，每个月我基本上都能通关数次，虽然后面增加了很多道具、玩法什么的，但我还是忠于基本模式，每天玩几局，权当消遣，尤其是现在卸载了王者荣耀之后。

推荐指数：⭐️⭐️⭐️⭐️★，做为一个微信小游戏，短时间获得正反馈，有一定的难度，已经很不容易了，推荐！
