产品的细节

发表于 2013-05-06   |   分类于 Thinking

好的产品是不断打磨出来的,就像一个工匠去雕琢一件艺术品一样.

最近几个月的时间我开始学习做饭,经常在厨房中碰到各种让我使用起来很不舒服的设计,比如:

  • 电饭煲外层和内胆的颜色是一模一样的,没有辨识度.某天早上我起床后晕晕乎乎的就将大米和水全部倒进了电饭煲,却没有注意到内胆被我放在一边.此时电饭煲的内部已经被大米和水堵了一个严严实实.我又花了一天的时间拆开电饭煲进行清理,还好没有多出几个零件.
  • 电饼铛的电源开关放在机器的后面,并且和电热盘离的很近.有一次我在使用完之后去关闭电源,食指碰到电源开关的时候,拇指也被电热盘烫伤了.之后每次使用电饼铛都是小心翼翼的.其实电源开关完全可以放在电饼铛的前面.
  • 马桶和洗碗池有死角,很难清理.马桶的底座要小于马桶的上围,所以底座和墙壁之间有一个死角,是最脏也最难清理的地方.洗碗池嵌在厨房台面的拐角处,同样有清理的死角.
  • 锅盖是玻璃的,外围用钢圈包了起来,接缝出同样很难清理.
  • 地铁上的宣传片中英文不对应,很容易造成误解.中文的句子为:”突发情况请拨打110或119”,大概是这么个意思.而英文的句子为:”Please call the number 110 119”.这里少了一个单词”or”,很多人都会理解为拨打号码110119.

阅读的工具

发表于 2013-05-01   |   分类于 Thinking

最近一年的时间买了很多的电子书,也越来越习惯用Kindle进行阅读.我的阅读习惯是:

  1. 第一遍通读,找到全书的框架和重点,初步判断这本书值不值得深入的阅读.
  2. 深入的阅读,多数情况下会使用脑图帮助自己整理整本书的结构.
  3. 做读书笔记,写下自己的思考.

我发现Kindle或者其他的电子设备更适合在第一遍阅读的时候使用,如果想要深入的阅读或者针对某一章节进行重点的注释,归纳和总结的话,这些电子设备使用起来并不是很顺手,经常有让人抓狂的现象.变现最突出的问题是:

  • 输入不方便,尤其是Kindle系列中没有键盘的那几款.
  • 注释没有一个集中管理的地方.
  • 页面间的跳转还是没有实体书方便,可能还是使用习惯的问题.
  • 现在大部分的电子书不管是目录还是排版都比较差

越大越好?

发表于 2013-04-26   |   分类于 Thinking

手机的配置越来越高了,从单核到双核,再到四核,内存,屏幕也越来也大,同时手机的容量也不断的攀升,上大学时有一个1G的TF卡已经很满足了,而现在手机的容量都是16G起.

同样的电脑的配置已经朝着怪兽的级别发展,但是运行的速度越没有感觉上的提升,原因是软件业在消耗着越多的硬件配置.不管硬件多么的高端,总是显得不够用.

移动网校的问题

发表于 2013-04-18   |   分类于 Thinking

移动网校有很多的细节问题一直都没有精力修改,这里我想说两个我不是很认同的问题,这两个问题背后反映的更多的是产品思路上的不同.

通关机制

网校的课程采取了通关的机制,即:用户购买课程之后,只能学习第一课,在满足一些条件之后,比如学习第一课的时间超过10分钟或者做题得分超过6分等.才可以解锁第二课,依此类推.
我是非常不赞成这种做法的,并提出过多次的建议,但每次都被以相同的理由给驳回:

  1. 这是我们的课程特色
  2. 用户的学习过程会充满乐趣(每次的解锁就是一种乐趣).
  3. 解锁机制可以促进用户不断的学习.

如果我们深入的去思考这个为题的话,会发现解锁机制并没有给我们带来如上的好处,相反却给我们造成了很多不必要的麻烦.

知识与经验

发表于 2013-04-16   |   分类于 Thinking

突然有一天发现自己的颈椎有了问题,查找颈椎病相关资料,引发颈椎病的原因主要有慢性劳损和受寒.这两样我都沾上了.突然想起我妈妈经常叮嘱我的:看电脑久了要活动一下,不要长时间的玩手机,躺下不要看书.而我之前仗着年轻,对妈妈的提示毫不在意,现在遇到了问题,才想起来其实很久之前我就收到过提示,如果能够早早注意这些问题的话,就不会出现颈椎劳损的情况了.

细细想来,在我们没有真正经历过一件事情的时候,通常对某些人告诉我们的”过来人经验”嗤之以鼻,直到经历之后才能真正的明白.就像高中的时候父母告诉我们要考上好的大学,长辈告诉我们年轻的时候不要挥霍身体.

为什么我们会忽略这些”过来人经验”呢?是因为叛逆的原因吗?不是,很多成年人同样会出现这样的问题.还是因为这些经验是错的?也不是,大部分的经验都是经过事实验证的,当然不排除一小部分特例.

我觉得关键原因在于知识和经验的区别.

摇滚青春

发表于 2013-04-13   |   分类于 Thinking

最开始喜欢摇滚实在高中二年级,那时候最喜欢听的是Coldplay,听他们的”Yellow”和”In my place”,后来还买了他们的专辑.之前虽然听过很过摇滚歌曲,比如Beyond,MJ等等,但对于摇滚到底是什么,却没有明确的概念.

大学的时候,开始听Queen的歌曲,随着听的摇滚越来越多,也慢慢改变了摇滚就是拿把吉他声竭力嘶的吼叫的看法.我在摇滚中听到了不一样的声音,是自由,是信仰,是独立的思想.

看微博上梁指导和”五月天”粉丝的各种黑,想想梁指导有一句话说的是非常正确的:唱功是确确实实存在的,每个歌手的唱功都有高低,但是你喜欢一个人,喜欢一首歌,并不仅仅因为他的唱功好坏.而是他可能代表了你某一时间段的心情,他陪你走过了一段困难的时期.就像Coldplay某一张专辑上写的那样:”摇滚像首诗“.
